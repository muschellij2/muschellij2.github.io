R packages

  • fslr: Wrapper functions for FSL (FMRIB Software Library) from Functional MRI of the Brain (FMRIB)
  • WhiteStripe: Whitestripe White Matter Normalization for Magnetic Resonance Images
  • brainR: Helper functions to misc3d and rgl packages for brain imaging
  • matlabr: R interface with calling MATLAB code without a server
  • spm12r: R interface with calling SPM12 MATLAB processing
  • rscopus: Interface with SCOPUS and Elsevier API in R
  • diffr: Display Differences Between Two Files using the Codediff JavaScript Library
  • drammsr: Port of Deformable Registration via Attribute Matching and Mutual-Saliency Weighting (DRAMMS) Registration to R
  • extrantsr: Additional functionality and extensions to the ANTsR R package
  • dcm2niir: R wrapper for dcm2nii DICOM converter
  • googleCite: Scraper for Google Citations
  • processVISION: Scripts for Parsing XML from VISION database
